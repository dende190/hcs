# hcs

to start the project you must have nodejs version 14.

Then you need to clone the project:

`git clone https://github.com/dende190/hcs.git`

next install the modules:

`npm install`

and copy file `.env.example` to `.env`

`cp .env.example .env`

## Run database

exec `npm start`

## Run send emails

exec `npm run email`

## Run Tests

exec `npm run test`
